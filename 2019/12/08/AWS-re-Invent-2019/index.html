AWS re:Invent 2019

Posted on 2019-12-08 17:57   |   In trends   |   views

自从AWS拉开云计算的序幕,它在改变IT架构和思路的路上一路绝尘,AWS每年召开的re:Invent大会也成为IT行业的风向标.每年AWS都会在re:Invent会上发布新的产品,很多产品都是颠覆性的,从S3,Redshift,DynamoDB到Lambda等等,他们创造了全新的概念并持续引领着行业的创新.IBM一向是只说不做,而AWS从来都是说完就是做完.看看今年AWS re:Invent有哪些新的产品.

日常自夸

简单开场之后,AWS CEO Andy Jassy开始了日常自夸.

  • Cloud Computing终于开始进入金融行业了,高盛的CEO不仅作为DJ出现在AWS re:Invent大会上,还做了AWS在高盛内部应用的演讲;
  • 全球有超过7000家政府机构使用AWS;
  • 很多Startup都使用AWS;

然后顺势引出了一个问题,在云计算的大势之下,企业如何reInvent新的业务模式? Uber,lyft,grab等改变了打车的习惯,Airbnb则改变了短租市场.Andy认为企业要在Cloud Computing环境中重构业务模式,需要做到以下几点:

  1. 确保高层管理人员能够与目标保持同步并找到解决问题的方法;
  2. 需要从上到下的进行激进的变革;
  3. 需要对基层员工进行有效的培训;
  4. 确保在迁移到云环境的过程中,业务不会被中断;

继续吹嘘AWS的产品在产品细节,深度和成熟度上完胜其他的云服务商(名副其实),然后开始发布新的产品.

新产品

Nitro系统

传统的hypervisor存在一个Dom0(特权域),通过Dom0能够操控hypervisor之上的虚拟机,这是一个隐患.AWS发布的nirtro系统将Dom0放置在专用的安全芯片中,提高了安全性.

Graviton芯片

发布新的ARM芯片,具有更好的经济效益.相比于X86系统,成本降低40%.

用于Inference的EC2类型

目前主流的ML系统都集中于优化Training过程,AWS提供新的ML实例,能够有效提高预测(predict)和推断(inference)的效率.系统采用了专用芯片,从而获得了10倍的lattency提升,3倍的吐吞,40%的成本降低;

AWS EKS/ AWS Farate

针对Container的管理问题,发布新的解决方案AWS Fargate.当然不忘吹嘘81%的container都运行在AWS之上,吊打友商,并且AWS的container管理集成了AWS其他所有服务(牛逼,AWS最可怕的地方就是所有的产品都是可以联动的).

Mainframe服务

这个地方有些模糊,是不是推出了mainframe服务?不过以AWS的惯性,推出mainframe服务也没啥.

Redshift更新

吐槽windows,吐槽微软授权策略.吹嘘IAM的安全性,吹嘘Redshift性能.Redshift能够并发扩展,能够根据查询SQL自动pre-computing,能够通过Spectrum服务查询S3,能够通过federated服务查询S3,Aurora等等;还能够和Datalake联动,并且按需付费,总之就是牛逼.

AUQA

发布新的Advanced Query Accelerator,更快的带宽,更快速的速度.

Elastic Search

发布Elastic Search服务.针对应用场景,能够根据查询请求,优化存储,达到更低的价格.当然不忘继续吐槽友商,我们的产品线就是丰富,我们不要拿个瑞士军刀就上战场,那都是过家家.专业的人用专业的DB干专业的事情!

UltraWarm Layer

能够根据请求,自动优化S3等底层存储

Cerner医疗公司与AWS合作

Cerner公司利用AWS的ML平台,减小医生诊断的Variance,利用NLP和语音技术提高医生效率.

Tensroflow/PyTorch/MXNet优化

优化AWS对TF,PyTroch,MXnet的支持,性能提高30%.

SegeMaker Studio

感觉就是定制化的JupyterNote服务,能够更好的分析数据,分析模型,评估模型,管理模型;

SegeMaker Autopilot

推出了全新的AutoML服务,利用各种牛逼的算法,自我迭代更新模型,一次性给你更多的选择.

GURU Code Review

终于对程序员下手了(前面一直在挤压程序员的生存空间),基于ML的CodeRiview系统,数据来自与AWS20年祖传代码,学习出来的模型能够快速发现你代码中的BUG.说你错就是错,AWS20年祖传代码就是牛逼!!!

Kendra索引服务

利用NLP技术,能够快速为第三方内容和文件建立有效的索引,快速进行搜索,支持现有的云存储(Dropbox等等);

Contact lens

智能客服?

AWS Ouputs

AWS集群托管服务,专门定制化的数据忠心,可以采用nativeAWS管理,也可以利用vmware管理.

AWS Local Zone

专门针对sigle-digit millisecond latency要求的服务.

结尾

  • AWS已经将ML用到了各个产品线中,未来应该还会扩大范围,优化系统性能;
  • AWS致力于简化ML的使用,这个过程还会持续,推动ML在商业系统中的应用;
  • Jupyter Notebook会成为新一代的交流工具,talk is cheap, show me the code, 各行各业都是这样子的.
  • ML产品正在拓展到各个领域,比如金融(高盛),医疗(Cerner);
  • AWS的产品深度和广度是其他的云服务商所难以达到的,会继续引领云计算市场;
  • AWS的产品也在借鉴其他厂商的思路,比如微软;
