问题

思路

null hypothesis

Posted on 2019-09-16 20:54   |   In algorithm   |   views

Null hypothesis又称为假设检验,是推论统计中用于检验统计假设的一种方法.而”统计假设”是可以通过观察一组随机变量的模型进行检验的科学假说.一旦能够估计未知参数,就能够根据结果对未知的真正参数作出适当的推论.统计上对参数的假设,就是对一个或者多个参数的论述,其中欲检验其正确性的为零假设(null hypothesis),零假设通常由研究者决定,反映了研究者对于未知参数的看法.相对于零假设的其他参数结论可以成为备择假设(alternative hypothesis),通常反应了执行检定的研究者对参数的另一种看法(通常也是以最希望知道的).
